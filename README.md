# 图片剪辑 · 字幕生成器

一个轻量的网页工具，用来在图片底部快速叠加多行字幕，并支持“切割层叠”效果。无需构建，直接打开即可使用。

## 功能特性
- 选择 PNG/JPG 图片并预览
- 多行字幕输入、自动统计行数（最多 20 行）
- 字幕高度、字体大小、字体样式/粗细/家族可调
- 字体填充色与描边色可选
- 浅色固定主题，页面四周浅紫色边框
- 字幕渲染逻辑：
  - 第 1 行：覆盖在原图底部
  - 第 2 行起：在原图下方连续叠放字幕条
  - 可在第 2 行起于字幕条下绘制与首行相同宽度的原图切片，形成层叠切割效果

## 使用说明
1. 打开 `index.html`（双击或拖入浏览器）
2. 选择图片
3. 在“字幕文本（多行）”输入框中按行输入字幕
4. 点击“生成字幕图”
5. 点击“保存图片”导出 PNG

## 本地预览（可选）
- 方式一：直接打开 `index.html`
- 方式二：启动本地服务器（需要安装 Python）

```bash
python -m http.server 8000 --bind 127.0.0.1
# 浏览器打开 http://127.0.0.1:8000/
```

## 文件结构
- `index.html` 页面结构与资源引用
- `styles.css` 样式与浅色主题
- `app.js` 字幕渲染与图片导出逻辑
- `prd.md` 需求描述（如有）

## 设计要点
- 画布尺寸随图片与字幕行数动态扩展
- 文本居中，超宽时自动缩放或省略号处理
- 字幕条背景为半透明黑色，增强可读性
- 兼容现代浏览器，无第三方依赖

## 常见问题
- 刷新后样式/脚本未更新：本项目对资源链接做了版本参数（如 `styles.css?v=20260209`），若仍有缓存问题可使用 Ctrl+F5 或无痕模式。
- 无法预览：直接打开 `index.html`，或按“本地预览”步骤启动服务器。
